HHA Review and Correct Report

HHA Review and Correct Report

The HHA Review and Correct report will allow providers to review their QM data to identify if there are any corrections or changes needed to the data prior to the quarter's data submission deadline. The report will provide a breakdown by measure and by quarter, of the HHA's assessment-based QM data for four rolling quarters. The report also identifies whether each quarter's data correction period is open or closed as of the report run date. The report output will also provide patient-level data in a CSV format the user can access via the user's My Reports page.
